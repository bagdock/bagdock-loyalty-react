# Changelog

## 0.1.0 (2026-04-04)

### Features

- Initial public release extracted from the Bagdock monorepo.
- `<BagdockLoyaltyProvider>` — configures the Loyalty client with theming.
- `<ReferralWidget>` — embeddable referral sharing widget.
- `<ReferralCard>` — individual referral card display.
- `<RewardsDisplay>` — shows available rewards.
- `<PointsDisplay>` — shows member point balance.
- `<LeaderboardWidget>` — referral leaderboard.
- `useLoyalty()` hook for custom integrations.
- Stripe/Clerk-style `appearance` prop with preset themes.
- Platform partner support with `platformApiKey` + `operatorId`.
